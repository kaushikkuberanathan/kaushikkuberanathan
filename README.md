## 🚀 Featured Project: Dugout Lineup

🌐 Live App: https://line-up-generator.vercel.app/  
📦 GitHub Repo: https://github.com/kaushikkuberanathan/lineup_generator  

A mobile-first lineup and game coordination platform for recreational baseball and softball teams.

Built to solve real problems coaches face:
- Generating **fair, development-focused lineups**
- Assigning **positions by inning**
- Managing **bench rotations**
- Sharing a **single source of truth** with parents and scorekeepers

**Why this matters:**
Most youth sports still rely on paper, memory, or fragmented tools.  
This project applies product thinking to a real-world coordination problem involving kids, development, fairness, and trust.
