## 🚀 Featured Project: Dugout Lineup

🌐 Live App: https://dugoutlineup.com
📦 GitHub Repo: https://github.com/kaushikkuberanathan/lineup_generator

A mobile-first lineup and game coordination platform for recreational baseball and softball teams.

Built to solve real problems coaches face:

* Generating **fair, development-focused lineups**
* Assigning **positions by inning**
* Managing **bench rotations**
* Supporting **game-day decision-making**
* Sharing a **single source of truth** with parents and scorekeepers

Built as a mobile-first PWA designed for real-time game-day use, with lineup generation, inning-by-inning position planning, bench rotation logic, and shareable views for parents and scorekeepers.

**Why this matters:** Most youth sports coordination still relies on paper, memory, spreadsheets, or fragmented tools. Dugout Lineup applies product thinking to a real-world coordination problem involving kids, development, fairness, trust, and time pressure.
